# 🛡️ Command Injection Vulnerability Assessment & Penetration Testing

## 📌 Overview
Performed VAPT on a Linux-based web application to identify command injection vulnerabilities.

---

## 🛠️ Tools Used
- Burp Suite
- Nmap
- Linux Terminal

---

## 🔎 Vulnerabilities Identified

- Blind Command Injection

---

## ⚠️ Impact
- Unauthorized system access
- Data exposure

---

## ✅ Mitigation

- Secure coding practices
